# MentorLM - универсальный ИИ-инструмент для студентов ВУЗов РФ

Веб-сервис с режимами ИИ-помощи, закрывающий различные типовые учебные задачи студентов. Цель — единый инструмент без необходимости сторонних сервисов.
Первый флагманский модуль - Бизнес-информатика.

## Режимы

1. "Общий" - универсальный режим для любых учебных вопросов, на базе ChatGPT 5.2
2. "Код" - специализированный режим для написания кода и его объяснения, на базе Claude Sonnet 4.5
3. "Модели" - специализированный режим для построения бизнес-моделей под любые задачи, любой сложности (Idef0, UML, EPC, BPMN, TOGAF)
